# Inteligencia Sevilla - Demos de IA

Este proyecto es una plataforma interactiva diseñada para **Inteligencia Sevilla**, una consultora especializada en la implementación de soluciones de Inteligencia Artificial para empresas. La web tiene como objetivo mostrar el potencial real de la IA a través de demostraciones en vivo.

## 🚀 Tecnologías Utilizadas

-   **React + Vite**: Para una experiencia de desarrollo ultra rápida y una aplicación web de alto rendimiento.
-   **Tailwind CSS**: Sistema de diseño basado en utilidades para un estilo premium y totalmente responsivo.
-   **GSAP (GreenSock Animation Platform)**: Utilizada para las animaciones cinemáticas del Hero, proporcionando una entrada visual impactante.
-   **Lucide React**: Set de iconos modernos y consistentes.
-   **TypeScript**: Para un desarrollo robusto y mantenible.

## ✨ Características Principales

### 1. Hero Premium (Cabecera)
- **Animaciones Cinemáticas**: Al cargar la página, el fondo realiza un suave zoom mientras los textos y botones aparecen con transiciones elegantes mediante GSAP.
- **Diseño Visual**: Fondo con modo de mezcla `screen` y gradientes de profundidad para un aspecto futurista y tecnológico.

### 2. Navegación Inteligente (Navbar)
- **Sticky & Glassmorphism**: El menú permanece fijo al hacer scroll, cambiando de transparente a un efecto de cristal desenfocado.
- **Detección de Secciones**: Resalta automáticamente la sección en la que se encuentra el usuario (Voz, WhatsApp, CRM o Contacto).
- **Desplazamiento Suave**: Los enlaces utilizan `scrollIntoView` para una navegación fluida sin recargas.

### 3. Demos Interactivas
La web está estructurada para guiar al usuario a través de tres demos clave:
- **Demo de Voz**: Automatización de llamadas y atención al cliente.
- **Demo de WhatsApp**: Gestión inteligente de mensajes y cualificación de leads.
- **Demo de CRM**: Integración y optimización de datos empresariales.

### 4. Conversión (CTA)
- Estratégicamente colocados botones de **"Auditoría Gratis"** y **"Solicitar Auditoría"** para filtrar leads interesados directamente en su consultoría.

## 🛠️ Cómo ejecutar el proyecto

1.  **Instalar dependencias**:
    ```bash
    npm install
    ```
2.  **Iniciar modo desarrollo**:
    ```bash
    npm run dev
    ```
3.  **Construir para producción**:
    ```bash
    npm run build
    ```

## 📁 Estructura del Proyecto

-   `/src/sections`: Contiene los componentes principales de cada sección de la página (Navbar, Hero/Header, Demos, Contacto).
-   `/src/components/ui`: Componentes base reutilizables (Botones, inputs, etc.) siguiendo el patrón de Shadcn/UI.
-   `/public/images`: Almacena los recursos visuales, incluyendo el fondo principal `hero_bg.png`.
